# StartupSense-AI

AI-Powered Startup Idea Validation and Analytics Dashboard built using Python, FastAPI, Streamlit, MongoDB, and Gemini AI.

## Features

* User Authentication
* Startup Idea Analysis
* AI SWOT Analysis
* Dashboard Analytics
* History Tracking
* MongoDB Database
* Downloadable Reports
* Plotly Charts
* Premium UI/UX

## Tech Stack

### Frontend

* Streamlit
* HTML/CSS
* Plotly

### Backend

* FastAPI
* Python
* Uvicorn

### Database

* MongoDB
* PyMongo

### AI

* Gemini AI API
* Rule-Based AI Fallback

## How to Run

### Activate Virtual Environment

```bash
.\venv\Scripts\activate
```

### Run Backend

```bash
uvicorn backend.main:app --reload
```

### Run Frontend

```bash
streamlit run frontend/app.py
```
